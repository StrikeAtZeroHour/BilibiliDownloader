It is a python program to download video from Bilibili. Execute the <executeMeToRun.py> to download videos.

--The downloaded video will be stored in C:\Users\user\Downloads\BilibiliSave

--Only support low resolution if you have no cookie to login Bilibili, as Bilibili only allow guest-user to watch low resolution video.

--You need to download ffmpeg and put it to PATH before use it.
